# About

The **String Operations Package** is a utility-focused Python package designed to streamline string handling in data processing tasks.

## 🔍 Why?

Text data often comes with accents, especially in languages like Portuguese, French, or Spanish. This package provides a simple way to normalize these strings, making them consistent and clean.
